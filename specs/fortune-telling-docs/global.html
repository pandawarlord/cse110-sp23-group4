displayFortunes()

This function uses getFortunes and displays the fortunes it gets on the page with a div element for each fortune. Every time this function is called, it clears everything in the element refered to by history and re-adds each fortune to the page. The date values of the fortunes are displayed according to the locale of the browser. So if your browser uses Spanish for the UI, the date would be in Spanish.
